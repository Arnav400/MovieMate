# MovieMate

Welcome to **MovieMate**, a sleek and modern movie directory application built with React, Vite, and Tailwind CSS. MovieMate leverages the TMDB (The Movie Database) API to provide users with an interactive way to explore movies, view details, and discover new favorites. Whether you're a cinephile or just looking for something to watch, MovieMate has you covered!

This project is hosted on GitHub Pages and can be accessed live at: [https://Arnav400.github.io/MovieMate](https://Arnav400.github.io/MovieMate).

## Features

- Browse a wide collection of movies fetched from the TMDB API.
- View detailed information about each movie, including title, synopsis, ratings, and more.
- Responsive design powered by Tailwind CSS, ensuring a seamless experience on all devices.
- Fast and optimized development experience with Vite.
- Persistent data storage and management using Appwrite (optional feature).
- Easy deployment to GitHub Pages with a streamlined build process.
